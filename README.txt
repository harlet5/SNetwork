# Social Network 
# Commands:
  - To run on your machine:
    1. Open 2 terminals
    2. In the first terminal run `go run .`
    3. in the second terminal `cd Frontend`, `npm install` and the `npm run dev`
    4. go to `localhost:8080`
  - To run in docker:
    0. Install docker and docker-compose if you haven't already
    1. run `docker-compose build` you might have to add `sudo` in front
    2. run `docker-compose up -d` you might have to add `sudo` in front
    3. go to `localhost:8080`
